# **_Patient-Doctor Telegram Bot_**


## Telegram bot that helps patients call for emergency care at the hospital and monitor doctors who are at shift

### Technical Stack:

- Java 17
- Spring Boot (v3.2.2)
- PostgreSQL
- Redis for Caching
- Liquibase Migrations
- RabbitMQ
- JUnit
- Docker
